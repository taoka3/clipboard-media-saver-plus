# Clipboard Media Saver Plus

WordPress 管理画面の **メディアライブラリ**で  
**メディアコピー → ペースト（Ctrl / ⌘ + V）するだけ**で  
画像・音声・動画をメディアライブラリへ保存できるプラグインです。

WordPress コアの **正規アップロード API のみ**を使用しており、
安全・軽量・管理画面専用の拡張機能として動作します。

---

## ✨ 特徴

- 📋 クリップボードから直接アップロード
- 🖼 画像 / 🔊 音声 / 🎬 動画に対応
- ⚙ WordPress 標準 API のみ使用
- 🔐 nonce / 権限チェック対応
- 🚀 メディアライブラリ画面限定で動作
- 🧩 常駐しない「機能追加型」プラグイン
